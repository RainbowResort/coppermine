Administration de Coppermine

Comme la plupart des applications qui ont une sorte de réglages pour l'utilisateur, Coppermine comprends des taches qui ne peuvent être faites que par un administrateur. Il est préférables de ne pas nommer le compte administrateur juste "admin" ou "administrateur", mais d'utiliser votre vrai nom ou un pseudo que vous voudrez voir apparaite à côté de vos téléchargements et de vos commentaires.
Il est indispensable que vous mémorisiez les données de votre compte administrateur et/ou les placiez à une endroit sécurisé; de même que vous pouvez régler votre navigateur pour qu'il mémorise le mot de passe, vous risuquez d'en avoir besoin plus tard.
Quels comptes ont des pouvoirs administrateurs ou non est déterminé par l'appartenance à un groupe d'utilisateur: les membres du groupe "administrators" ont les droits d'administration. Vous pouvez gérer vos groupes dans le "panneau de gestion des groupes".
Techniquement, il est possible d'avoir plus d'un administrateur de la galerie, mais il est recommandé d'être prudent avant de donner les droits d'administration à d'autres: il ne faut donner ces pouvoirs d'administration qu'a des personnes de confiance totale.

Identifié comme administrateur, vous verrez un menu administrateur affichant des options complémentaires et des entrées de menu que vous (en tant qu'admin) avez comparé aux utilisateurs normaux (non admin) ou aux visiteurs.

Les réglages les plus basiques aussi bien que les réglages avancés ayant un impact sur l'ensemble de votre galerie sont définis dans la page de configuration - certains sont obligatoires pour paramétrer correctement votre galerie lors d'une première utilisation, il est donc recommendé de regarder attentivement la section configuration .

Dans Coppermine, les fichiers sont organisés dans une structure de catégories/albums. Pour paramétrer les catégories et les albums et pour leur maintenance, utilisez le panneau de gestion des catégories et le géstionnaire d'Albums.

Votre galerie Coppermine est capable d'extraire les méta données des images et de les stocker dans la base de donnée. Vous pouvez gérer les données au format EXIF en utilisant le gestionnaire d'EXIF.

Des fonctions complémentaires qui ne sont pas necessaires pour toutes les galeries peuvent être ajoutées en utilisant des plugins.

Coppermine peut appliquer un filigrane (watermark) aux images téléchargées afin de minimiser le vol de contenu.

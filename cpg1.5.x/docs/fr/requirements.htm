Qu'est ce qui est requis

Il y a certains minimum requis nécessaires pour l'installation de Coppermine:

  • Un serveur Web supportant PHP
    PHP (version 4.3.0 ou plus récente), compilée avec le support pour la librairie graphique GD library ou la permission d'utiliser la fonction exec() function pour l'utilitaire "convert" d'ImageMagick afin de pouvoir créer les vignettes et les images intermédiaires. le support de Mysql doit aussi être compilé dans PHP.
  • une base de donnée MySQL
    Vous avez besoin de MySQL version 3.23.23 ou plus récent. MySQL 4.1 est recommendé, MySQL 5 est supporté. L'utilisateur MySQL doit avoir les permissions pour créer (CREATE), modifier (ALTER), remplacer (REPLACE), délectionner (SELECT), mettre à jour (UPDATE), effacer (DELETE), insérer (INSERT) des données dans la base de donnée. La plupart des utilisateurs ont ces permissions avec les serveurs web. Si ce n'est pas le cas, vous ne pourez pas Coppermine ni aucun script. Si vous n'êtes pas sur d'avoir ces permissions, vérifiez avec votre hébergeur. La plupart des hébergeurs pourront certainement vous dire si Coppermine peu tourner sur leurs serveurs.
    Une base de donnée doit être crée afin que Coppermine puisse l'utiliser - le script d'installation ne crée pas de base de donnée pour vous, mais va créer automatiquement les tables et la structure des donnée dans votre base de donnée pendant l'installation. Dans la plupart des cas avec un hébergeur, vous aurez déjà une base de donnée créer par votre hébergeur. Sinon, il vous faudra la créer. Si vous devez créer une base de donnée, regardez avec votr ehébergeur quelle est la bonne methode pour le faire. Notez le nom de votre base de donnée, votre Nom d'utilisateur et le mots de passe. Il vous faudra les trois pour installer correctement Coppermine.
  • Une librairie graphique:
    Soit GD version 1.x ou 2.x (PHP doit être compilé pour ouvoir l'utiliser) ou ImageMagick.
    (regardez Qu'est ce qu'ImageMagick et Qu'est ce que GD)
  • C'est certainement une bonne idée de contacter en premier votre hébergeur pour savoir si ils sont au courrant de problèmes avec l'installation de Coppermine sur leurs serveurs.

Donc Coppermine peut fonctionner sur tous les seveurs web qui ont les minimums requis mentionnés plus haut, L'équipe de développement de Coppermine ne peut pas vous aider pour configurer votre serveur. Comme indiqué, il y a un minimum requis à vérifier et à mettre en place avant d'installer Coppermine. Notez que l'équipe de Coppermine re recommande pas l'hébergement personnel sauf si vous avez une bonne expérience dans l'administration de serveurs web et que vous savez comment faire.

Requis personnels

Coppermine est destiné aux webmasters qui ont au moins un savoir faire de base en HTML et une certaine expérience dans la mise en place de scripts. L'équipe de dévelopement de Coppermine sait que certains utilisateur qui ont peu ou pas d'expérience dans ce domaine utilisent les auto-installeurs pour configurer leur galerie, ce qui est très bien. Nous vous demandons simplement de lire attentivement cette documentation avant de poser des questions sur le forum d'aide de Coppermine, la plupart des réponses aux questions de débutants s'y trouvent. Vous pouvez utiliser Coppermine sans problèmes avec les themes qui se trouvent dans le pack. Néanmoins, si vous voulez modifierun theme existant ou crére votr epropre theme (qui correspond à l'aspect du freste de votre site), vous devrez avoir des connaissance de base voir avancées en HTML et CSS .

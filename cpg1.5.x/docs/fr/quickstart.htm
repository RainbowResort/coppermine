Question: Mon serveur supporte t'il Coppermine?
Réponse: Lisez la section sur le minimum requis.

Question: Ou dois-je mettre tous ces fichiers?
Réponse: Vous pouvez les mettre dans le répertoire racine (non recommandé) ou les placer dans un répertoire Coppermine de votre site web. Si vous n'êtes pas sur de savoir exactement dans quel répertoire le faire, demandez à votre hébergeur dans quel répertoire vous devez télécharger vos fichiers html et autres fichiers de script. Nous vous recommandons de télécharger le répertoire cpgXXXX (et tous les fichiers qu'il contient) directement sur votre serveur dans le répertoire recommandé par votre hébergeur, et de le renommer comme vous le souhaitez après le téléchargement complet. Comparez le contenu du pack de Copermine avec les fichiers téléchargés. Assurez vous que tous le fichiers ont été téléchargés correctement.

Question: Ok, j'ai téléchargé tout sur www.monsite.com/cpgXXX. Et maintenant?
Réponse: Vous devez changer les permissions d'un certain nombre de répertoires avant de faire quoi que ce soit d'autre. Utilisez votre client FTP ou le Panneau de controle de votre site pour changer les droits pour les répertoires: "include", "albums", "albums/userpics", et "albums/edit" à 777. Vous devrez aussi créer un nouveau sous-répertoire dans le répertoire albums dans lequel vous placerez vos images/fichiers par FTP (nommez le "uploads" ou de la manière que vous souhaitez - simplement n'y mettez pas d'espaces ou d'autres caractères spéciaux.) et paramétrez aussi les permissions à '777'. Sur certains serveurs, cela ne fonctionnera pas vace un chmod à '777'. Dans ce cas, utilisez '755'. (Perdu? vous ne savez pas ce qu'est le CHMOD? Vous avez été prévenus, devons nous vous dire autre chose?)

Question: J'ai changé les permission. La suite?
Réponse: Vous devez créer une base de donéne Mysql. Utilisez le panneau de controle de votre site ou un outil comme PhpMyAdmin et créez la base de donnée. Notez le nom de la base de donnée, le nom d'utilisateur et le mot de passe - vous en aurez besoin lors de l'installation de Coppermine Coppermine. Si vous avez déjà une base de donnée utilisée par un forum ou une autre application, celà fonctionnera aussi. Vous pourez aussi utiliser cette base de donnée pour Coppermine. Si vous avez un forum et que vous planifiez de le connecter à Coppermine, vous devrez de toute manière partager la base de donnée.

Question: Mes fichiers sont en place, les droits sint corrects et j'ai les informations concernant ma base de donnée MySQL. Quelle est la prochaine étape?
Réponse: Lisez la... (Oops, pardon.)  Maintenant vous devez lancer l'installation de Coppermine. Ouvrez votre navigateur web et entrez http://www.votresite.com/votre_repertoire_coppermine/install.php et entrez les informations demandées. Si vus n'avez jamais utilisé ImageMagik avant, laissez cette ligne vide pendant le processus d'installation. Vous pourrez toujours ajouter ces informations par la suite, une fois l'installation terminée et lorsque votre galerie Coppermine fonctionnera correctement.

Question: Ha! Ha! Je l'ai eu! C'est installé et ça marche! et maintenant?
Réponse: 1) Effacez ce sourire de votre visage. 2) Lisez la documentation, encore -- une deuxième fois. 3) Vous devrez regarder la configuration de Coppermine pour être sur que la galerie est parametrée comme vous le souhaitez. Identifiez vous comme administrateur si vous n'êtes pas encore identifié. Cliquez sur le bouton Configuration. Il y a plusieurs choses auquelles vous devez faire attention:

  • Paramètres Généraux: Vous devrez entrer le nom de votre site, et votre adresse courriel ainsi que votre zone horaire GMT.
  • Paramétrage des Langues, Themes & Jeux de Caractères: Choisissez votre langue par défaut.
  • Paramétrages des Fichiers et Vignettes: Définissez la taille maximale pour le téléchargement des fichiers que vous autorisez (en Kilobytes). Le paramétrage par défaut est 1024kb. Assurez vous que cette taille maximale est plus petite ou égale à celle autorisée par votre hébergeur. Tant que vous y êtes, vous devriez aussi définir la "Largeur ou hauteur maximale pour le téléchargement des images/vidéos (pixels) téléchargés" - Celà empèchera vos visiteurs à télécharger des images plus grandes que ce que vous souhaitez. Si vous souhaitez que Coppermine redimmensionne automatiquement les images plus grandes que ce que vous avez défini dans la configuration, paramétrez "Redimenssionne automatiquement les images plus garndes que la largeur ou hauteur maximale" sur soit "Oui: Tout le Monde" ou "Oui: Utilisateur uniquement". "Tout le monde" signifie tout le monde, y compris lorsque l'administrateur (vous) téléchargez les images par lot, et "Utilisateurs" ne s'applique qu'aux membres de votre galerie lorsqu'ils téléchargent des images.
  • Paramètre avancés pour les fichiers et les vignettes: Assurez vous que vous avez sélectionné le bon programme de manipulation d'images. Si vous utilisez GD, Vus devriez sélectionner GD2 si il est disponible sur votre serveur. Si vous souhaitez utiliser ImageMagik, vous devrez entrer le chemin vers le programme(assurez vous que le chemin se termine par un slash de fin "/"). Voulez vous permettre aux membre de créer des albums privés? Si oui, assurez vous d'avoir paramétré "Les utilisateurs peuvent avoir des albums privés" sur "Oui".
  • Paramétrage des utilisateurs: Si vous souhaitez laisser les visiteurs s'enregistrer et télécharger des images, assurez vous que "Permettre les nouvelles inscriptions" est paramétré à "Oui".
  • Vous devrez aussi jetter un oeil aux paramétrages des groupes. Coppermine a trois groupes par défaut: administrateurs, Utilisateurs Enregistrés, Utilisateurs Annonymes. Vous trouverez ici les controles pour les permissions d'envoi des E-Cartes, le téléchargement de fichiers etc. C'est aussi là que vous pourrez paramétrer l'espace disque maximum alloué pour chaque groupe (quota disque). Il est paramétré à 1024 Kb par défaut.

Question: Je pense que tout est paramétré correctement. Je peux télécharger les images maintenant?
Réponse: Non. Vous devez d'abbord créer un album pour les y placer. Dans la même ligne que le lien pour la Configuration vous verrez un bouton ou un lien "Albums". Cliquez dessus et vous serz ammené à un écran qui vou spermettra de créer un album. Cliquez sur "Nouvel Album", puis renommé le comme vous le souhaitez. Cliquez sur "Appliquer les changements" et voilà, vous avez un album maintenant.

Question: J'ai créer un album, alors je peux maintenant télécharger mes images?
Réponse: Oui. En tant qu'administrateur, vous devriez utilier un client FTP pour télécharger vos images sur votre serveur, puis faites un ajout par lot pour les enregistre dan svotre base de donnée Coppermine. Au début e ce guide, je vous demandais de créer un sous-répertoire dans le répertoire "Albums" justement dans cet objectif. Admettons que vous l'ayez nommé "uploads". Placez vos images par FTP dans votre répertoire albums/uploads. Une fois terminé, cliquez sir le bouton FTP. Selectionnez "uploads" comme répertoire. Vous avez maintenant devant vous une liste de tous les fichiers que vous venez de télécharger. Par défaut, Coppermine sélectionne toutes le images. Selectionnez l'album dans lequel vous voulez placer les images puis cliquez sur "envoyer". Vous avez maintenant une galerie en ligne avec vos images visibles! Incroyable, Non ?

Question: Maintenant, je peux ajouter un titre à une image?
Réponse: Oui vous pouvez. Sur l'écran principal de Coppermine, en étant identifié comme administrateur, vous verrez une liste d'options à côté de l'album. Si vous cliquez sur "éditer les fichiers", vous pourrez ajouter toutes sortes d'informations pour chaque image.

Question: Je peux effacer l'image originale maintenant Non? Après tout, elles sont dans la base de donnée, c'est çà ?
Réponse: Faux! Coppermine ne place actuellement pas les images dans la base de donnée. Il utilise la base de donnée uniquement pour se rappeler ou les images sont stockées sur le serveur, quel album et quel catégorie lui sont attachés ainsi que les informations relatives à l'image. Si vous effacez les images originales, vous aurez une série d'erreurs. Si vous voulez effacer les images originales, cherchez dans la documentation a quoi servent les outils Administrateurs et ce pourquoi ils sont fait. Vous ne devirez jamais effacer quoi que ce soit en utilisant les outils de votre hébergeur ou votr eclient FTP si cela peut être fait directement par Coppermine. En le faisant, vous aurez des erreurs sur votre serveur et dans votre base de donnée MySQL.

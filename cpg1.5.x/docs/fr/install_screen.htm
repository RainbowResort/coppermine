Table des Matières

L'écran d'installation

La version cpg1.5.x change beaucoup des versions précédentes et la methode d'installation est particulièrement concernée: toutes les versions de Coppermine ont un installeur, mais dans cpg1.5.x, le processus d'installation à été entièrement refait: l'installeur maintenant par défaut a été pourvu d'une interface semblable à un assistant, ou vous pouvez faire une étape à la fois. Vous serez accompagnés pendant l'ensemble du processus d'installation en utilisant l'assistant, ce qui est une grande avancée pour les utilisateurs débutants. Néanmoins, si vous êtes un expert de Coppermine et que vous savez ce que vous faites avec cette application, vous pouvez toujours utiliser l'ancien écrant d'installation: nous l'avons inclu dans le pack sous le nom de "classic installer". Merci de noter qu'il n'y aura pas d'aide pour cette methode: si vous avez des problèmes lors de l'installation et que vous voulez de l'aide, utilisez le nouvel installeur. Les questions relatives à l'ancien installeur resteront sans réponses sur le forum de Coppermine. Bien entendu, l'ancien installeur ne comprends pas toutes les fonctions du nouvel installeur: il ne vrée pas la base de donnée pour vous etc...
Ceci dit, voic vos options:
  • l'écran d'installation classique
  • Le (nouvel) assistant d'installation (recommandé)

L'écran d'installation classique

Your admin account (Votre compte Administrateur)
Cette section demande les informations pour la création de votre compte administrateur. N'entrez que des caratères alphanumériques. Entrez les données avec soin ! Entrez ici un nom d'administrateur et un mot de passe. Assurez vous de bien les mémoriser, ou notez les dans un endroit sur; vous ne pourrez plus administrer votre galerie si vous perdez ces données.
Username (Nom d'Utilisateur) Ce sera le nom d'utilisateur pour l'administration quotidienne de Coppermine - choisissez en un que vous pourrez mémoriser et entrer facilement. Cette saisie est sensible à la casse, soyez donc particulièrement attentif lors de la création du nom d'utilisateur. Les images que vous téléchargerez plus tard sous ce compte administrateur afficheront ce nom dans le champ "Téléchargé par". Les commentaires que vous ferez afficheront aussi ce nom visble par les autres.
Password (Mot de Passe) Ce sera votre mot de passe administrateur pourvotre installation de Coppermine. N'utilisez pas de mots de passes évidents et trop souvent utilisés - si un hacker trouve votre mot de passe, il/elle pourra hacker l'ensemble de votre site! Utilisez une combinaison de majuscules et de miniscules avec des chiffres comme "j3e4N5n6yG". Rappelez vous, les mots de passe tout comme votre nom d'utilisateur administrateur sont sesibles à la casse. Soyez prudents lors de la cration du mot de passe. Copiez le et palcez le dans un endroit sur autant que possible loint de votre ordinateur
Email address (Adresse Courriel) Cette adress courriel sera utilisée pour envoyer des courriels depuis le serveur web (exemples:le courriel d'enregistrement, les notifications et les e-cartes). Assurez vous que ce soit une adress courriel valide.
Your MySQL configuration (votre configuration MySQL) MySQL est le type de service de base de donnée disponible sur la plupart des serveurs web. Si vous ne l'avez pas, vous ne pouvez pas l'installer vous-même. Si vous avez un hébegement web, vous êtes probablement malchanceux. Il est obligatoir d'avoir une base de donnée MySQL pour utiliser Coppermine ou beaucoup d'autres scipts PHP. Vous ne pouvez pas truquer les informations MySQL pendant l'installation, vous devez les connaitre avant, et entrer les informations comme demandé. Si vous n'êtes pas sur des informations requises, demandez votre hébergeur.
Cette partie requière les informations pour se connecter à votre base de donnée MySQL. Si vous ne savez pas comment la remplir, vérifiez avec l'aide de votre hébergeur.
MySQL Host (Hôte MySQL)
(localhost est souvent OK)
MySQL Database Name (Nom de la base de donnée MySQL) Coppermine ne va pas créer cette base de donnée pour vous - elle doit exister avant 'essayer d'installer Coppermine. (Si vous n'en de base de donnée pré établie, vous devrez en créer une, à condition d'avoir le droit de le faire pour votre site).
MySQL Username (Nom d'Utilisateur MySQL) Le nom d'utilisateur MySQL ne doit pas être votre nom d'administrateur Coppermine, ni être nécessairement le même que votre nom d'utilisateur FTP (quoique cela peut être le cas pour certains utilisateurs, mais uniquement par simple coïncidence ou de manière délibérée, pour simplifier l'administration du site).
L'utilisateur MySQL doit avoir les permissions pour les fonctions CREATE TABLE, INSERT, ALTER, REPLACE, UPDATE, DELETE, SELECT.
MySQL Password (Mot de passe MySQL) Le Mot de Passe qui va avec le nom d'utilisateur MySQL.
MySQL table prefix (préfixe des tables MySQL)
(La valeur par défaut est bonne; si vous deviez en changer, n'utilisez pas de point!) 		Les tables de Coppermine peuvent coéxister avec les tables d'une autre application dans la base de donnée. Sauf si vous savez ce que vous faites, ne changez pas la valeur par défaut.
ImageMagick Vous ne pouvez pas truquer un chemin d'accès à ImageMagick, vous devez le connaitre. Si vous n'êtes pas sur, laissez ce champs vide - Coppermine essayera d'utiliser GD par défaut. Vous poyrez modifier le chemin d'accès plus tard dans la page de configuration. Vous ne pourrez installer ImageMagick ou GD que si vous pouvez adminsitrer entièrement votre serveur - si vous êtes hébergés par un hébergeur web, vous ne pourrez certainement pas. L'utilisateur qui souhaite utiliser ImageMagick, devra avoir des droits d'accès en lecture/écriture/et éxécution sur le répertoire ou se trouve l'éxecutable "convert" d'ImageMagik.
Coppermine peut utiliser le programme "convert" d'ImageMagick pour créer les vignettes. La qualité des images produites par ImageMagick est supérieure à GD1 mais équivalente à.

Si ImageMagick est installé sur votre système, et que vous voulez l'utiliser, vous devrez entrer le chemin complet vers le programme "convert" ici. Sur Windows le chemin ressemble à 'c:/ImageMagick/' (utilisez / pas \ dans le chemin d'accès) et ne doit pas contenir d'espaces, sur Unix ce sera quelque chose comme '/usr/bin/X11/'.

L'assistant d'installation

Depuis cpg1.5.x Coppermine utilise un assistant d'installation qui vous guide à travers la totalité du processus d'installation. Ci-dessous vous trouverez les étapes de l'assistant d'installation, avec un chapitre par étape.
N'utilisez pas les boutons précédent/suivant de votre navigateur si vous voullez revenir en arrière ou aller en avant dans l'installeur. A la place: n'utilisez que les boutons de controles inclus dans l'installeur. Vous aurez des résultats imprévisibles si vous n'écoutez pas ce conseil. Pour revenir en arrière d'une étape, utilisez les liens en haut à droite de l'installeur ou toutes les étapes que vous avez déjà franchies sont surlignées et cliquables.

License agrément & sélection de la langue

Sur la page d'accueil de l'assistant vous verrez la license sous laquelle est publié Coppermine. Si vous n'acceptez pas cette licence, vous n'êtes pas autorisés à continuer l'assistant d'installation. Vous pourrez relire la licence par la suite (après avoir terminé l'installation) en allant dans cette documentation et en allant à la section "Documentation de Copperine" → "License & Copyright".
Sous la licence, vous trouverez un sélecteur qui vous permets de choisir votre langue préférée pour l'installation. Généralement votr elanque est détectée automatiquement, vous ne devirez pas avoir à faire quoi que ce soit ici - cliquez juste sur le bouton "Allons-y" tout en bas de la page d'accueil pour continuer l'assistant.

Vérification des fichiers d'installation

A la seconde étape de l'installation, un evérification est faite pour voir si le pack que vous voullez installer est le plus récent. Ceci est fait afin de vous assurer que vous êtes bien en train d'installer la dernière version de Coppermine, dans laquelle tous les bugs connus sont corrigés. Généralement, si vous venez juste de télécharger les fichiers depuis le site officiel de Coppermine sur sourceforge.net, tout va bien. Si l'installeur vous indique que vous n'avez pas la dernière version, vous devriez télécherger celle-ci avant de continuer.
Cliquez sur "étape suivante" si tous vos fichiers sont à jour ou si vous ne souhaitez pas avoir la dernière version.

Vérification des droits des répertoires

Coppermine doit avoir les droits en écriture dans certains sou-répertoires comme expliqué dans Paramètres des droits. L'installeur ne peut pas définir ces droits pour vous, vous devez doc le faire vous même auparavant. Si vous l'avez fait correctement, tous les répertoires vérifiés par l'asistant d'installation devraient être marqués "OK". Si ce n'est pas le cas pour vous ( si l'installeur vous signale des manques pour un ou plusieurs répertoires), laissez l'écran du navigateur comme il est et allez paramétrer les droits correctement. Une fois fait, revenez à l'cran d'installation et rafraichissez la page pour voir si le problème des droits est résolu.
Vous ne pourrez pas terminer l'installation correctement si les droits ne sont pas définis correctement: l'installeur doit pouvoir écrire les données de configuration ue vous allez entrer dans un fichier. Si l'installeur n'y arrive pas à causse de mauvais droits, vous ne pourrez pas terminer l'installation.

Sélection de la librairie graphique

Coppermine à besoin d'une librairie graphique présente sur le serveur (voir minimum requis → librairie graphique). Actuellmement GD et ImageMagick sont supportés. A cette étape de l'assistant d'installation, plusieurs vérifications sont faites pour définir quelle librairie graphique est disponible sur votre serveur. Généralement, vous pouvez aller à l'étape suivant si il n'y a qu'une librairie graphique trouvée. Si l'installeur indique qu'il ne trouve pas de librairire graphique, vous devrez manuellement la définir. Vous ne pouvez pas passer cette étape, Coppermine n'installera pas de librairie graphique pour vous - c'est une partie des minimum requis. Si vous n'avez pas une des librairie graphique supportée par Coppermine, vous ne pourrez pas utiliser le script.

Test de la librairie graphique

A cette étape, la librairie graphique que vous avez sélectionné précédemment va être testée: l'installeur va essayer de réaliser cetaines opérations comme le redimensionnement ou la réation d'image que le script de Coppermine utilisera après l'avoir configuré. Généralement le test se passe bien et vous verrez les images générées dans la colonne de gauche. Si c'est bien le cas pour vous, vous pouvez co,tinuer sans craintes. Si ce n'est pas le cas, vous devrez revenir en arrière d'une étape et choisir une autre librairie graphique. Une autre raison possible de ce difonctionnement peut aussi être un droit manquant.

Autentification de l'utilisateur MySQL

Pendant cette étape, vous devrez entrer vos données de connection MySQL. Vous ne pouvez pas falsifier les informations MySQL pendant l'installation, vous devez les connaitre avant et entrer les informations comme demandé. Si vous n'êtes pas sur de la valeur des informations nécéssaires, demandez votre hébergeur. L'équipe de développement de Coppermine ne sera pas en mesure de vous dire ce que vous devrez entrer à cet endroit - vous devrez demander à vore hébergeur pour celà.
Vous devez définir
  • Le nom d'hôteMySQL
    localhost est généralement bon pour la plupart des configurationsis usually OK on most setups
  • Le nom d'utilisateur MySQL
  • Le mot de passe
Juste sous les champs de saisie, vous trouverez un bouton pour tester la connection - si vous cliquez sur ce bouton, un test est effectué pour vérifier que les valeurs que vous avez entré sont correctes. Si c'est le cas, vous pouvez passer à l'étape suivante. Sinon, vous devrez revoir les paramètres d'autentification MySQL.

Sélection de la base de donnée MySQL

Ici, vous devrez choisir la base de donnée que vous souhaitez utiliser pour Coppermine.
Si votre compte MySQL a les privilèges nécéssaires, vous pouvez créer une nouvelle base de donnée depuis l'installeur, ou utiliser une base de donnée existante. Si vous n'aimez aucune de ces deux solutions, vous devrez crée d'abord une base de donnée en dehors de l'installeur de Coppermine, puis revenir ici et sélectionner la nouvelle base de donnée dans la liste déroulante.
Vous pouvez aussi changer le préfixe des tables (n'utilisez pas de points), mais il est recommandé de laisser le paramètre par défaut.
Pour installer plus d'une galerie Coppermine dans une même base de donnée, utilisez des préfixes de tables différents.
Après avoir sélectionné vos préférences, cliquez sur le bouton "remplir la base de donnée".

Création de la structure de la base de donnée

L'écran de structure de la base de donnée n'est là que pour votr einformation - vous ne pourez agir sur rien ici. L'écra ontr ela progression des opérations réalisées pour créer les tables de la base de donnée de votr egalerie. Si tout se passe bien, vous verrez une série de lignes avec des messages du type Création de a table 'XXX' Status:... OK. Si vous n'avezpas le status "oK" à chaque ligne, quelque chose ne s'est pas bien passé et il faudra vérifier. C'est généralement le cas si le compte utilisateur MySQL que vous avez défini à l'étape "Autentification de l'utilisateur MySQL" n'a pas les privilèges nécessaires à la création de tables. Une autre raison d'erreur peut provenir d'un manque de place dans la base de donnée. Dans les deux cas, vous devrez contacter votre hébergeur pour avoir de l'aide.
Si tout c'est bien passé, vou spouvez passer à la prochaine étape de l'installeur en cliquant sur le bouton en bas de cet écran.

Création de l'administrateur de Coppermine

Pendant cette étape, vous devrez créer un compte utilisateur avec les droits d'administration pour vous. Il est très important d'entrer les données très attentivement. Quelques trucs qu'il faut se rappeler:
  • N'utlisez que des caractères alphanumériques. Pas d'espaces ni de caractères spéciaux!
  • Utilisez une adresse couriel existante - au cas ou vous oublieriez votre mot de passe, vous pourrez demander à ce qu'un nouveau mot de passe soit envoyé à cette adresse.
  • Mémorisez votr enom d'utlisateur et votr emot de passe - vous feriez bien de le mettre à un endroit sur (l'équipe de développement de Coppermine recommande d'utiliser une application tièrce comme KeePass pour gérer vos mots de passe et les stocker en sécurité).
  • Il est recommandé de ne pas utiliser de noms d'utilisateurs génériques comme "admin" ou "administrateur", parce qu'ils peuvent être facilement devinés et permettront des attaques automatique de forces brutes plus facilement. Le nom de l'administrateur apparait à plusieurs endroits de l'interface utilisateur de Coppermine pour toutes les actions que vous ferez en tant qu'administrateur (comme le téléchargement, les commentaires etc...).
  • Choisissez votre mot de passe attentivement - il ne doit pas être trop court, et n edoit pas être commun. L'équipe de développement de Coppermine recommande d'utiliser un ephrase qui veut dire quelque chose pour vous et d'utiliser la première lettre de chaque mot pour votre mot de passe. Idéallement le mot de passe doit contenir des majuscules et des minuscules ainsi que des nombres. Par exemple regardez cette phrase "Chaque fois que les 5 chiens de mon voisin me voient Je dois courir" - elle donnera comme mot de passe "Cfql5c2mvmvJdc"
Après avoir rempli le champ comme demandé, cliquez le bouton en bas de page.

Installation terminée

Une fois que vous verrez l'écra final, vous aurrez réussi - félcitation. Cliquez sur le bouton en bas de page ou sur n'importe quel lien qui vous amènera à la page d'accueil de votr egalerie ou à la page d'identification ou vous pourrez vous identifier en tant qu'administrateur pour configurer votre galerie.
Nous vous recommandons de continuer par la lecture la section de la documentation Pour commencer maintenant.

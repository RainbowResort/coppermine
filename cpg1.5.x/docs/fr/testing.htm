Version Beta (Test)

Cher testeur,

Merci de prendre du temps pour tester Coppermine Photo Gallery. Merci de malmener le programme afin de tracker et d'éliminer les bugs de la version finale publique de cpg 1.5.

Comprenez qu'il s'agit de'une version Béta du script qui ne doit pas être utilisée en production mais pour des tests, il n'y a pas de support d'aide pour cpg1.5.x pour l'instant.

Pendant vos tests de CPG 1.5.x, prenez soins de noter tous les disfonctionnements que vous rencontrerez. Que ce soient des erreurs typographiques, des instructions trompeuses ou toute autre erreur. Si vous rencontrez une errer majeure, placez le mode de débuggage sur "on" dans le panneau de configuration et copiez le texte de la fenêtr ede débbogage. Celà aidera l'équipe de développement à résoudre le problème.

Pas de Support pour les versions alpha et beta:

L'Equipe de Coppermine est prête à dispenser une aide sur le forum officiel de Coppermine pour les versions qui ont été officiellement été mises à disposition. C'est pourquoi il n'y a pas de support pour les versions qui sont au stade alpha ou beta , c'est a dire en développement. Seules les versions stables ont droit à un support d'aide.

Si vous avez besoin d'aide, vous dvez utiliser la version stable la plus récente.

Bien entendu vous êtes le bienvenue pour poster des rapports de test si vous utilisez une version encore en développement. Toutefois il y a une limite très fine entre un rapport de test valide et une demande d'aide: si vous postez quelque chose en rapport avec une version en développement et que votre rapport montre clairement que vous n'avez même pas pris la peine de lire la documentation fournie, et cheché sur le forum avant, vous recevrez une réponse sévère ou pas de réponse du tout.

Domaines demandant une attention particulière:

  • Merci de tester chaque facette de CPG, de l'installation à chaque fonction disponible (et ce dans le plus grand nombre de themes et de langues possible).
  • Si vous avez la possibilité de tester GD1, GD2 et ImageMagick, merci de tous les tester
  • Merci d'essayer d'utiliser plusieurs types de fichiers graphiques comme JPEG, PNG et GIF (pourle utilisateurs d'ImageMagick et de GD 2.0.28+).

Merci de rapporter le résulat de vos tests dans le forum de tests, mais avant de commencer un nouveau sujet, assurez vous que le bug potentiel que vous avez découvert n'a pas déjà été rapporté avant.

Le raport devrait inclure:

  • Le système d'exploitation du serveur
  • Le logiciel du serveur
  • Si le test a eu lieu avec GD, GD2 ou ImageMagick
  • La version de PHP
  • L'intégration avec un forum (si c'est le cas)
  • Les themes entierement testés (par entièrement testé, nous entendons le plus de fonctions différentes possibles en utilisant les themes)
  • Les erreurs majeurs
  • Votre système d'exploitation et votre Navigateur Internet
  • Si il s'agit d'une nouvelle installation ou d'une mise à jour (dans ce cas depuis quelle version de cpg)

Le résultat de vos tests ne servira pas uniquement à régler les éventuels problèmes, mais à vérifier la compatibilité de avec un maximum de systèmes d'exploitation et de navigateurs. C'est pourquoi ces informations sont importantes, MÊME SI VOUS N'AVEZ PAS TROUVE D'ERREURS.

Merci pour votre participation et le temps que vous avez donné pour aider Coppermine à être la première galerie photo Open-Source!

ATTENTION: cpg1.5.x n'a pas encore été mis à la disposition du public - seule une récupération sur le svn est actuellement possible (dossier cpg1.5.x). Lors du test de versions alpha, assurez vous que votr eversion est bien la plus récente.

L'équipe de développement de Coppermine

Documentation du développeur

Cible

Cette partie de la documentation n'est pas destinée aux utilisateurs finaux, mais uniquement pour les développeurs. Il n'y a pas d'aide pour cette section, elle n'est là telle-qu'elle.

Contenu

La documentation pour les développeurs est loin d'être complête. Néanmoins voci les sections qui existent actuellement. Comme cette documentation est en constante évolution, regardez aussi la documentation en anglais qui est peut être plus complête:

  • Guide de codage
    A lire obligatoirement pour les membres de l'équipe de développement de Coppermine, sa lecture est recommandée pour les (potentiels) contributeurs
  • Ajout/renomage/effacement de fichiers
    Instructions pour les développeurs expliquant ce qui doit être fait lors de la création de nouveaux fichiers dans le noyaux de Coppermine
  • Sécurisation des Superglobales en utilisant Inspekt
    Pour comprendre comment la sécurisation fonctionne et ce que vous devez faire pour que les Superglobales fonctionnent correctement
  • Javascript dans Coppermine
  • Ecrire des Plugins pour Coppermine
    Pour les auteurs de Plugins
  • Modifier la documentation
    Pour les développeurs et les traducteurs
  • Subversion
    Besoin de la version la plus récente du SVN ? Regardez ici.
  • Ajout d'options de configuration
    Les développeurs qui veulent ajouter des options de configuration doivent lire cette section.
  • Variables & Constantes
    Un tour d'horizon des variables globales utilisées par Coppermine. Très utile si vous êtes un programmeur et que vous voulez comprendre comment Coppermine fonctionne pour pouvoir être en mesure d'utiliser les variables dans votre propre code (par exemple si vous êtes un auteur de plugin)
  • Vérification de version

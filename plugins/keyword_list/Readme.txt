Keywold List plugin
Add a menu button to link to a new page.
On the top, all the keywords like a search menu.
clicking on a keyword display the title from all  pictures who have this keyword.
On mouse over the title, a preview pop up window.


INSTALL:
- unzip package to plugins folder and use plugin manager to install it


CONFIG:



LANGUAGE:
english.php and french.php are avaible in the package. You can translate in your language

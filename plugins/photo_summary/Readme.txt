Photo Summary plugin by Frantz
Displaying an file index page
original MOD by fotofreek http://coppermine-gallery.net/forum/index.php?topic=4419.0


INSTALL:
unzip package to plugins folder and use  the plugin manager to install it

    
CAUTION:
to work properly, your pictures must minimum have a title. If not, you can use "Updates titles from filename" feature from Admin utilities. 


LANGUAGE:
english.php, italian and french.php are avaible in the package. You can translate in your language

Was ist es?

Dieses Plugin fügt einen Link hinzu, damit Besucher das Bild in voller Größe herunterladen können.

Installation

Das Plugin wird über den Coppermine-Plugin Manager installiert. Es sind keine gesonderten Schritte notwendig.

Deinstallation

Deinstallation erfolgt über den Plugin-Manager.

Konfiguration

  1. Den Download-Link in Kategorien zeigen, die diesem regulären Ausdruck entsprechen

    Wenn der Download-Link nicht überall gezeigt werden soll, sondern nur in bestimmten Kategorien, dann kann das mit dieser Einstellung erreicht werden. Es können reguläre Ausdrücke gemäßt der PHP-Funktion preg_match() verwendet werden, oder wer sich damit nicht so gut auskennt, kann zum Beispiel /.*/ verwenden für alle Kategorien oder etwas in der Art wie /^(Kategorie 1|Kat 2|Noch eine)$/ um bestimmte Kategorien aufzulisten.

  2. Den Link nur angemeldeten Benutzern anzeigen

    Ja bedeutet, dass nicht angemeldete Benutzer keinen Download-Link zu sehen kriegen. Nein bedeutet, dass alle Benutzer den Download-Link sehen können.

  3. Den Link in der Tabelle mit den Bildinformationen zeigen

    Ja bedeutet, dass der Download-Link in der Tabelle mit den Bildinformationen unter dem Bild angezeigt wird. Nein bedeutet, dass der Download-Link als Button unter des Bild oder in den Titel oder in die Beschreibung mithineingebastelt wird.

  4. Direkt abspeichern (mit Content-Disposition)

    Ja bedeutet, dass der Browser das Bild direkt herunterläd. Im Hintergrund verwendet dies den "Content-Disposition"-Header und sollte mit jedem modernen Browser funktionieren. Nein bedeutet, dass das Bild in voller Größe in einem einzelnen neuen Browser-Fenster angezeigt wird.

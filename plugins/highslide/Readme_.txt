Very first Highslide Plugin v 3.04
  Based on HighSlide.js orginaly writen by "Torstein Hnsi" @ www.vikjavev.no/megsjol

  Skip Intermediate Page and show full page on the page

Installation:
1. Uninstall any previous version.
2. Delete settings table if asked.
3. Copy Highslide folder to your plugins folder.
4. Copy hscnt.php to your gallery root.
5. Use plugin manager to install it.

Please read Changelog file for features and known issues.
If you have found any bug post it to the announcement thread.

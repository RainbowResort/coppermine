What it does:
=============
This plugin can be used to make avatar from all the image in the gallery. 
User can set avatar width & height, can choose where to crop the image
and maybe add some effect (in future versions).
*********************************************

How to install:
===============
Install this plugin as you would install any other plugin (as suggested in
the documentation that comes with coppermine).

Plugins for the Coppermine Photo Gallery are add-ons that use the plugin interface of Coppermine.
Read up the documentation that comes with Coppermine to find out more, e.g. how to install a plugin.

Coppermine version: cpg1.5.x
Plugin name: Filetype editor
Plugin version: 2.0
Plugin author: Nibbler
Plugin announcement thread: http://forum.coppermine-gallery.net/index.php/topic,59917.0.html


This plugin provides the admin with an easy to use interface to Coppermine's filetypes table, avoiding the need to use phpmyadmin. You can add, edit and delete file types and set the default player for each type of file.

Plugin für CPG 1.5.x: Image Manipulation

Worum geht's?

Dieses PlugIn ermöglicht es dem Besucher, non-destruktiv die Bilder auf der Seite displayimage.php zu verändern. Je nach Konfiguration des PlugIns werden die Einstellungen für jedes Bild in einem Cookie auf dem Computer des Besuchers gespeichert (besucht er dasselbe Bild später erneut, so sind seine Einstellungen wieder da), und die URL wird dynamisch angepasst (gibt der Besucher also den Link an jemand anders weiter, so sieht der das Bild mit dem Einstellungen, die er bevorzugte). Dieses PlugIn fügt etwa 30 KByte an Javascript zur Seite hinzu.

Einstellungen

Kompatiblen Modus benutzen

Verfügbare Effekte im Kompatibilitätsmodus oder falls der Besucher den IE als Browser benutzt

  • Helligkeit (LED-Regler)
  • Original
  • Weichzeichnen
  • S/W
  • Negativ
  • Relief
  • Spiegeln horizontal
  • Spiegeln vertikal

Verfügbare Effekte mit abgeschalteten Kompatibilitätsmodus und falls der Besucher nicht den IE als Browser benutzt

  • Helligkeit (LED-Regler)
  • Kontrast (LED-Regler)
  • Sättigung (LED-Regler)
  • Schärfe (LED-Regler)
  • Original
  • Sepia
  • Spiegeln vertikal
  • Spiegeln horizontal
  • Negativ
  • Relief
  • Weichzeichnen

Browser-Kompatibilität

Die Effekte funktionieren mit
  • IE 5.5+
  • Opera 9.5+
  • Firefox 2+
  • Chrome 3+
  • die meisten anderen Browser sollten ebenfalls funktionieren oder zumindest problemlos weiterarbeiten

Bitte beachten

  • Dieses PlugIn funktioniert nur dann, wenn 'Transparente Grafik über dem tatsächlichen Bild einfügen, um Bilderdiebstahl zu reduzieren' in den Galerie-Einstellungen deaktiviert ist.
  • Dieses PlugIn und das PlugIn 'mirror' dürfen nicht zugleich benutzt werden.
  • Die neuen Schaltflächen sind nur für Benutzer verfügbar, die Javascript aktiviert haben. Alle anderen sehen nichts.
  • Es ist empfohlen, den Kompatibilitätsmodus zu benutzen. Tust du das nicht, so hängt es vom Browser des Besuchers ab, welche Effekte verfügbar sind.
  • Benutze die Konfigurations-Seite, die du über den Plugin-Manager erreichst, um Kompatibilitätsmodus, die Benutzung von Cookies und URL-Werten einzustellen.

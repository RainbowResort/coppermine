Plugin for CPG 1.5.x: JSmin

What's this?

Using cpg1.5.x, many javascript files will be loaded by the visitor's browser. There are four files that will always be loaded on every page (jquery-1.3.2.js, scripts.js, jquery.greybox.js, jquery.elastic.js). More files will be loaded depending on the actual page. As long as you're not logged in as administrator, this plugin does the following:

  • Step 1:
  • merge the files jquery-1.3.2.js, scripts.js, jquery.greybox.js and jquery.elastic.js to one javascript file
  • compress it using JSmin to remove comments and disposable spaces
  • if browser supports it, compress it with gzip
  • cache the resulting file so compression has only to be done once
  • Step 2:
  • merge all other javascript files of the current page to another file
  • compress this one using JSmin to remove comments and disposable spaces
  • if browser supports it, compress it with gzip
  • cache the resulting file so compression has only to be done once

Result

Every page of your gallery will no longer include 5 to 20 different javascript files, but only two of them. This way, there will only be two HTTP requests going to the server because of javascript files. This should speed up access significantly.
The size of the javascript files will decrease 50-80% for browsers that support zlib functions, and 10-15% for browsers that don't. This will save bandwidth and download time.
There's nearly no additional server load, because once the compressed files are created, they will be cached forever (as long as you do not clear the cache manually).

maintenance

Whenever you update your gallery, install additional plugins, update existing plugins or apply mods you must clear the cache to make your changes work. Please use the button on plugin manager page to do this. If you forget this, your changes will work for the administrator, but not for guests or registered users.

Plugins for the Coppermine Photo Gallery are add-ons that use the plugin interface of Coppermine.
Read up the documentation that comes with Coppermine to find out more, e.g. how to install a plugin.

Coppermine version: cpg1.5.x
Plugin name: Keyboard Navigation
Plugin version: 1.1.1
Plugin author: eenemeenemuu
Plugin announcement thread: http://forum.coppermine-gallery.net/index.php/topic,57471.0.html

With this plugin you can navigate to the previous/next picture, go back to the thumbnail page and switch the file info details via the keyboard arrows.
